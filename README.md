1. null এবং undefined এর মধ্যে পার্থক্য কী?

undefined মানে হলো কোনো ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু তাকে এখনো কোনো ভ্যালু দেওয়া হয়নি। অর্থাৎ JavaScript নিজে থেকে সেটাকে undefined করে দেয়।

আর null হলো ইচ্ছাকৃতভাবে দেওয়া একটি খালি ভ্যালু। মানে নিজ ইচ্ছায় খালি ভ্যালু সেট করা।

2. map() ফাংশনের কাজ কী? এটি forEach() থেকে আলাদা কীভাবে?

map() একটি array এর প্রতিটি উপাদানের উপর কাজ করে এবং নতুন একটি array তৈরি করে। অর্থাৎ এটি original array পরিবর্তন না করে একটি নতুন array রিটার্ন করে।

অন্যদিকে forEach() শুধু প্রতিটি উপাদানের উপর কাজ চালায়, কিন্তু কোনো নতুন array রিটার্ন করে না।

3. == এবং === এর মধ্যে পার্থক্য কী?

== (double equal) শুধু ভ্যালু compare করে। এটি data type convert করে নেয়।

=== (triple equal) ভ্যালু এবং data type দুটোই compare করে। এখানে কোনো type conversion হয় না।

4. API data fetch করার ক্ষেত্রে async/await এর গুরুত্ব কী?

async/await ব্যবহার করলে asynchronous কাজগুলো সহজ ও পরিষ্কারভাবে লেখা যায়। API থেকে ডেটা আনা একটি asynchronous কাজ, কারণ ডেটা আসতে সময় লাগে।

আগে Promise chain ব্যবহার করলে কোড জটিল হয়ে যেত। কিন্তু async/await ব্যবহার করলে কোড দেখতে synchronous এর মতো লাগে, ফলে পড়া এবং বোঝা সহজ হয়।

5. JavaScript এ Scope কী? (Global, Function, Block)

Scope মানে হলো — একটি ভ্যারিয়েবল কে কোন কোন জায়গা থেকে / কোথায় থেকে অ্যাক্সেস করা যাবে।

Global Scope:
যে ভ্যারিয়েবল পুরো ফাইল বা প্রোগ্রাম জুড়ে ব্যবহার করা যায়।

Function Scope:
যে ভ্যারিয়েবল কোনো ফাংশনের ভিতরে ডিক্লেয়ার করা হয়, সেটি শুধু সেই ফাংশনের ভিতরেই ব্যবহার করা যায়।

Block Scope:
যে ভ্যারিয়েবল {} এর ভিতরে (যেমন if, loop ইত্যাদি) ডিক্লেয়ার করা হয়, সেটি শুধু ওই ব্লকের মধ্যেই সীমাবদ্ধ থাকে।
